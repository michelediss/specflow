# Specflow Developer Skill

Daily usage:

1. Read `specflow/architect/spec/SPEC.md` and `specflow/architect/decisions/DECISIONS.md` for context.
2. Pick the relevant `T-xx` task.
3. Implement tests and the minimal code to satisfy the linked ACs.
4. Run `npm --prefix specflow run dev:guard -- --mode=developer` and `npm --prefix app test`.
